idea集成tomcat服务器

idea集成tomcat服务器：

  1. 利用idea创建一个普通java工程

  2. 右单击项目名，菜单点击添加框架支持

  3. 选中WebApplication

会产生一个web文件，里面有一个默认起始的页面

  1. 点击添加配置

  2. +号

Tomcat Server ->本地

  1. 按照这个页面设置

  2. 添加一个工件

  4. 确认后尝试运行一下
